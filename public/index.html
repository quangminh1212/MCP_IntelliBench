MCP Tools

6 powerful tools for comprehensive AI coding evaluation

intellibench_start_session

Start a new benchmark session with customizable categories and difficulty.

intellibench_get_challenge

Get the current challenge with description, requirements, and code template.

intellibench_submit_solution

Submit solution code for evaluation and scoring across all dimensions.

intellibench_get_results

Get comprehensive results with scores, strengths, and recommendations.

intellibench_list_challenges

Browse available challenges filtered by category and difficulty.

intellibench_leaderboard
